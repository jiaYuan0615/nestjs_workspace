## Description

NestJS application with Domain-Driven-Design Workspace

## Enviroment Version

```
Version
 - node 16.17.0
 - npm 8.15.0
```

## Installation

```bash
$ cp .env.example .env
$ npm install
```

## Running the app

```bash
# development
$ npm start
```

## Building the app

```bash

# Using Origin Webpack
$ npm run webpack

# Using Nest with Webpack
$ npm run build

# Start up the build app
$ node ./dist/main.js
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
